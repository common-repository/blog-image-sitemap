Sitemap for blog pictures.

== Description ==

Create sitemap for blog pictures with cron schedule.

= Requirements =

* WordPress 5.0 or higher

== Installation ==

Unzip the image-sitemap folder to your WordPress plugins folder. It should work out of the box.
